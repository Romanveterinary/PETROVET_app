# 🛡️ PETROVET — Цифровий Ревізор (v49.0)

Професійний інструмент для ветеринарно-санітарної експертизи на ринках України.

### ⚖️ Юридичний фундамент програми:
1. **Наказ №16** — Положення про державну лабораторію вет-сан експертизи.
2. **Наказ №57/188/84/105** — Правила торгівлі на ринках (ваги, чистота).
3. **Наказ №46** — Інструкція з клеймування м'яса.
4. **Наказ №28** — Правила вет-сан експертизи м'яса та субпродуктів.
5. **Закон №1870-IV** — Про молоко та молочні продукти (заборона ПЕТ-тари).
6. **Наказ №1032** — Заборона домашньої ковбаси та консервації грибів.
7. **Закон №771** — Вимоги до маркування та кулінарних виробів.
8. **Наказ №185** — Зберігання круп та спецій (герметичність).
9. **Наказ №455** — Експертиза рослинної продукції.
10. **Наказ №23** — Експертиза риби та рибних продуктів.
11. **КУпАП ст. 107, 160, 166-22** — Адміністративні штрафи за порушення.
